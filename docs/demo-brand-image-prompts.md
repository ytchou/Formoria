## Generation Notes

- **Style consistency:** All images for a single brand should share a consistent color temperature and editing style. Run each brand's images in a batch to maintain visual cohesion.
- **Taiwanese aesthetic:** Favor warm, natural tones. Avoid overly saturated or artificial-looking images. Think Kinfolk magazine meets Taiwanese craft culture.
- **Background cleanup:** After generation, ensure all product photos have clean, distraction-free backgrounds suitable for e-commerce display.
- **File naming convention:** `{brand-slug}/{type}-{number}.{ext}` (e.g., `shan-wu-tea-house/product-1.jpg`)
- **Upload path:** Supabase Storage bucket `brand-assets`, folder per brand slug.
